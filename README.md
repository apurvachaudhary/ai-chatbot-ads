# AI Chatbot for Google Ads Automation

This project automates Google Ads campaign creation based on business inputs.

## Setup

### Backend
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the backend:
   ```bash
   uvicorn main:app --reload
   ```

### Frontend
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the frontend:
   ```bash
   npm start
   ```
